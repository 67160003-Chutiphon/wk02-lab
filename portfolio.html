ผลงานของฉัน

โปรเจกต์ที่ 1: E-Commerce Website

พัฒนาด้วย React, Node.js และ MongoDB ระบบมีการจัดการสินค้า ตะกร้า และระบบชำระเงินแบบครบวงจร

โปรเจกต์ที่ 2: Blog Platform

สร้างด้วย Next.js และ Prisma มีระบบการเขียนบทความ แสดงความคิดเห็น และระบบค้นหาที่รวดเร็ว

โปรเจกต์ที่ 3: Portfolio Website

ออกแบบด้วย HTML5, CSS3 และ JavaScript vanilla มีการใช้ animation และ responsive design ที่ทำงานได้ดีทุกอุปกรณ์
